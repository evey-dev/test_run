SLIDE 7 - SAE AND INTERVENTION MATHEMATICS                            1:35
========================================================================

TITLE:
  The learned basis and the exact edit

ON-SCREEN TEXT:
  u_l = x_l / s_l
  a_l = ReLU(W_e(u_l - b_d) + b_e)
  z_l = TopK_k(a_l)
  u_hat_l = W_d z_l + b_d

  Inhibit S:  x'_l = x_l - s_l W_{d,S} z_{l,S}
  Swap S:     x'_l = x_l + s_l W_{d,S}(z^src_{l,S} - z^tgt_{l,S})

FIGURE:
  `presentation/figures/fig_presentation_sae_and_intervention.pdf`

SPOKEN SCRIPT:
  "Let x sub l denote the MLP output at layer l and the edited token position,
  and let s sub l be its training scale. The encoder maps the normalised output
  into eight thousand one hundred and ninety-two candidate latents. Top-K keeps
  exactly k non-zero coordinates, and the decoder reconstructs the activation
  from unit-normalised feature directions. To inhibit a selected set S, I
  subtract only those decoded contributions from the original MLP output. To
  swap, I add the difference between the source and target feature
  contributions. In both cases the unselected part of the original activation
  remains intact. The resulting operation is an error-preserving feature edit
  applied directly to the original model's MLP output."

SLIDE 8 - ATTRIBUTION GRAPH                                          1:20
========================================================================

TITLE:
  Contrastive attribution ranks candidates for causal testing

ON-SCREEN TEXT:
  J(x) = ell_target(x) - ell_contrast(x)

  A_lj = z_lj * (partial J / partial z_lj)

  Arithmetic example: J = ell_4 - ell_3

  Straight-through splice:
  exact model output in the forward pass; SAE path in the backward pass.

FIGURE:
  `report/figures/fig_math_attribution_graph.pdf`

SPOKEN SCRIPT:
  "A logit, written ell, is the unnormalised score assigned to a candidate next
  token before the softmax. I analyse a contrast J: the target logit minus a
  pre-specified alternative. The first-order attribution of feature j at layer
  l is its activation multiplied by the gradient of J with respect to that
  activation. Positive values locally support the target side of the contrast.
  I use a straight-through splice so that the forward value remains the exact
  model activation while gradients pass through the SAE representation. This
  keeps reconstruction error out of the forward baseline. The resulting pruned
  graph supplies ranked candidates for disjoint causal confirmation."

SLIDE 9 - EVIDENCE STANDARD                                          1:15
========================================================================

TITLE:
  Three complementary levels of evidence

ON-SCREEN TEXT:
  Reconstruction
  FVE = 1 - E[||x - x_hat||^2] / E[||x - mean(x)||^2]

  Decodability
  Can a held-out classifier recover the condition from the features?

  Causal specificity
  Does a frozen edit change the target contrast more than its matched control?

  Mechanistic claim = causal specificity on disjoint confirmation data

FIGURE:
  `presentation/figures/fig_presentation_evidence_standard.pdf`

SPOKEN SCRIPT:
  "The central methodological distinction is shown here. Fraction of variance
  explained asks whether the SAE reconstructs its input. Held-out decodability
  asks whether the representation contains information about a condition. A
  causal test asks whether editing a frozen feature set changes the model's
  behaviour selectively. These levels answer different questions: fidelity,
  information content and causal use. A representation can retain almost all
  variance while distributing one computation across many latents, and a
  decodable label may be a consequence or correlate of the computation.
  Controlled intervention therefore supplies the decisive mechanistic test."

BACKUP A - SYMBOLS AND INTERVENTION DEFINITIONS

TITLE:
  Mathematical definitions

ON-SCREEN TEXT:
  x_l       MLP output at layer l and edited token position
  s_l       per-layer activation scale estimated on training data
  z_lj      activation of SAE feature j at layer l
  W_d[:,j]  decoder direction for feature j
  ell_t     pre-softmax logit for token t

  p(t|x) = exp(ell_t) / sum_v exp(ell_v)
  J = ell_target - ell_contrast
  A_lj = z_lj * partial J / partial z_lj

FIGURE:
  `presentation/figures/fig_presentation_sae_and_intervention.pdf`

SPOKEN SCRIPT:
  "A logit is the model's unnormalised score for a token. Softmax converts the
  vector of logits into probabilities. I intervene on the MLP output x at a
  chosen layer and position. Each z coordinate weights one decoder direction.
